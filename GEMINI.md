# agentic-ui-testing

This repo is intended to provide a hands-on demo and walkthrough of executing automated UI tests from Gemini CLI. The demo will be consumable as a Google Codelab.

It is based on my blog [Creating an Automated UI Test of Your Web App in Seconds with Gemini CLI and BrowserMCP](https://medium.com/google-cloud/creating-an-automated-ui-test-of-your-web-app-in-seconds-with-gemini-cli-and-browsermcp-09cf4afb8940)

## Structure of the Demo / Codelab

1. Problem statement: the challenge of creating and running UI tests.
1. Our demo application.
1. Intro to MCP as a way to intelligently run tools from your AI agent, by supplying natural language prompts.
1. Intro to BrowserMCP as a way to run browser tools from your AI agent.
1. Running BrowserMCP via Gemini CLI to create and run an automated UI test.
1. Now let's do it with Playwright with a skill.
1. You can do this in Antigravity!
1. Other use cases for browser control.
